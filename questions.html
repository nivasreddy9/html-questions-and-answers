3.what is the diff b/w href attr and src attr?

href:

* href stands for HyperText Reference
* href is used to refer to another HTML element
* href is used to refer to pages on the same website

src:

* SRC stands for Source Reference.
* SRC refers to a non-HTML file.
* SRC refers to pages on different servers.
